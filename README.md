# 數位影像處理概論

### 影像處理程式開發平台3

**預計新增功能**

- 人臉偵測 : 先使用別人訓練好的(.xml)模組。
- 口罩偵測 : 尋找如何自己訓練(.xml)模組訓練。

---

**新增功能以下功能:**

- 找尋輪廓

- 邊緣與輪廓

- 膨脹(dilation)與侵蝕(erosion)


---

**已經做出功能以下功能:**

- 開啟圖檔
- ROI
- 低通濾波器
  - 平滑模糊
  - 高斯模糊
  - 中值濾波器
  - 雙邊濾波器
- 高通濾波器
  - 拉普拉斯
  - Sobel
  - Canny
- 仿射轉換
  - 旋轉
  - 平移
  - 翻轉
- 透視投影
- Gray & HSV
- Binary_Value
- 自訂參數的仿射轉換
